# Seppmail-API-PowerShell

> PowerShell-centric access patterns for the SEPPmail REST API.

**Partner resource by Synedat for the SEPPmail ecosystem.**

## Why this repository exists

Documentation-first repository for PowerShell usage around the SEPPmail API, with links to current official API references.

This repository is structured for public consumption and easy discovery across topics such as SEPPmail, Exchange Online, Microsoft 365, Azure, API automation, PowerShell and operational runbooks.

## Quick start

1. Enable and scope API access on the SEPPmail system before testing.
2. Use token-based access and store secrets securely.
3. Prefer current official API docs whenever endpoints or permissions change.
